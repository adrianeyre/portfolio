## <a name="Install">Installation</a>
* To clone the repo
```shell
$ git clone https://github.com/adrianeyre/portfolio
$ cd portfolio
$ npm install --global yarn
$ yarn
```

## <a name="Tests">Running the Tests</a>
```shell
$ npm run test
```

## <a name="Run">Running the Website</a>
```shell
$ npm start
```
